### 结构目录
例子:

  - 2015届 // 届别
    - XX作业 // 作业标题
      - 姓名 // 文件夹
        - files
      
### 首次提交流程
  1. fork本仓库
  
  2. 克隆仓库至本地 `git clone https://github.com/your-name/Homework.git`
  
  3. **按照示例格式添加作业文件**
  
  
  4. 提交修改到暂存区 `git add .`
  
  5. 提交修改到本地库 `git commit -m "your-commit-info"`
  
  6.提交修改到远程库 `git push`
  
  7.通过github网站发起pull request
